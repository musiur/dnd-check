## Troubleshooting Visual Issues

### Cards Not Dragging
- Check that `draggable="true"` is set
- Verify `onDragStart` handler is attached

### No Drop Indicator
- Ensure `onDragOver` calls `e.preventDefault()`
- Check that drop zone has proper handlers

### No Visual Feedback
- Verify CSS classes are applied
- Check Tailwind CSS is working
- Inspect element styles in DevTools

### Animations Not Working
- DND-KIT animations only work within Kanban
- Native drags don't have built-in animations
- Check that `transition` styles are applied
